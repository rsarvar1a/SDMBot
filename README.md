# SDM Bot

## Usage

```sh
git clone git@github.com:rsarvar1a/SDMBot.git
cd SDMBot
python main.py
```

To set up your configuration file, provide a `config.json` in the root directory
(**DO NOT COMMIT THIS FILE**):

```json
{
  "token": "<DISCORD TOKEN>",
  "prefix": "s!",
  "admins": [
    "<USER ID>",
    "<USER ID>",
    ...
  ]
}
```
